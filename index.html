Workbook Schema Reference
Dealer Target Fy27 · FY27 Dump · FY26 Dump · ROI (optional, for Partner Financial Health)
📋 Dealer Target Fy27
Phone No · Dealer Name · Location · ASM Name · Region · Zone · FY27 Sales AOP · Monthly actuals (Apr-26 → Mar-27) · FY26 Actuals
📊 FY27 Dump (actual sales — rupees ÷ 100,000 = lakhs)
Month · Dealer Name · Mobile No · Final basic Value ( without Inst) · Territory · Final Mapping · ASM Name · ASM EMP CODE · City · State · Dealer/Direct · Segment
📊 FY26 Dump (actual sales — rupees ÷ 100,000 = lakhs)
Month · Dealer Phone · Dealer Name · Final basic Value ( without Inst) · Territory · Final Mapping · ASM Name · ASM EMP CODE · City · State · Dealer/Direct
💹 ROI / Partner Health (optional)
Accepts either the pre-computed Partner Health Data sheet (Operating Profit, PBIT, ROI, Health Status already calculated) or a raw Q1-FY27 style sheet — Dealer name · Phone No · Customer price · Sales Margin · TOD · Showroom Making Cost · Display Material Cost · Age in months · Depreciation · Manpower & Overheads (monthly sub-fields). Sheet name can be "ROI", "Partner Health Data", or "Q1 - FYxx".
📐 Formula Reference
Total Income = Sales Margin + TOD · Opex = (Monthly Manpower + Overheads) × Expense Months · Operating Profit = Income − Opex · PBIT = Operating Profit − Depreciation · ROI = Operating Profit ÷ CAPEX Net Book Value (CAPEX depreciated straight-line over 120 months) · Health Status: Critical (PBIT<0) → Watch (PBDIT%<5%) → Stable (<10%) → Healthy.
