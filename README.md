# Channel Intelligence - PWA Companion Files

## Setup
1. Extract this ZIP into your GitHub repo (same folder as index.html)
2. Push to GitHub and enable GitHub Pages (Settings > Pages > main branch / root)
3. Done! Install prompt activates automatically on HTTPS.

## Folder structure
your-repo/
  index.html
  manifest.webmanifest
  sw.js
  icons/
    icon-72.png ... icon-512.png

## Notes
- Android Chrome: shows Install App banner automatically
- iPhone Safari: Share > Add to Home Screen
- If your HTML is not named index.html, update start_url in manifest.webmanifest
